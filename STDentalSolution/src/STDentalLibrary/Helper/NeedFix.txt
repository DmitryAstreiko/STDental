1. при удалении пациента, добавить проверку на существование пациентов в талонах
2. при удалении единицы измерения, добавить проверку на существование ед. изм в материалах, прейскуранте
3. при удалении материала, добавить проверку на существование материала в прейскуранте
4. изменить тип данных в БД у полей StartTime, EndTime у таблицы Receptions (т.к. у MS SQL 2014 через EF не понимает тип данных time, time(7))
5. обновление материалов и пересчет прейскуранта при изменение материала
6. при удалении прейскуранта проверить, что он не находится в талоне - CheckContainServiceInTalons
